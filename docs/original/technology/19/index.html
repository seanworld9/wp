WordPress環境構築@Sakura VPS

2023年10月13日

SakuraVPS の OSインストール

https://kusanagi.tokyo/cloud/kusanagi-for-sakuravps/

マルチサイト構築も可能

# kusanagi provision siteA　（Asite.com)
# kusanagi provision siteB　（Bsite.net）

これで同一ホスト（IP）に複数ドメインを割り当てることが可能になる。
ルーティング（Virtualhost）も自動で書き込んでくれているっぽい。すごい。

テーマのインストール

https://wp-cocoon.com/

コクーンというテーマが無難そう。スキンを変えるだけである程度設定も可能。

必要なファイル編集

wp-config.php
define(\’FS_METHOD\’, \’direct\’);

/home/kusanagi/siteA/DocumentRoot/wp-content/themes
の権限設定を行う

<pre class="wp-block-preformatted"># chmod 744 themes
