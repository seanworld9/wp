]]> SSH待ち受けポートの変更手順【CentOS】 http://seanworld.net:10214/original/technology/94/ Fri, 13 Oct 2023 06:10:37 +0000 https://seanworld.net/?p=94 CentOSログイン時、不穏なメッセージが表示された。

There were *** failed login attempts since the last successful login

これは、前回ログイン（成功）時から、***回ログインに失敗したよ、というメッセージである。ポートをうっかりデフォルトの22番のままにしてしまっていたことが原因である。早速変えたいと思う。ちなみに、過去に不正にログイン試行されたログはこの記事の手順で調べられる。

SSHDのコンフィグを変更

 # vi /etc/ssh/sshd_config
 - # Port 22
 + Port 6022
 # systemctl restart sshd

実はこれだけではだめなのです。（Ubuntuとかはこれだけで良かった気がしますが）

Firewallの設定も変更

# firewall-cmd --permanent --remove-service=ssh  //SSHのファイアウォールを削除
# cp /usr/lib/firewalld/services/ssh.xml /etc/firewalld/services/ssh-6022.xml //XMLファイルを作成
# vi /etc/firewalld/services/ssh-6022.xml //portを変更
 -  <port protocol="tcp" port="22"/>
 +  <port protocol="tcp" port="6022"/>
# firewall-cmd --permanent --add-service=ssh-6022 // 6022のサービスを読み込む
# firewall-cmd --reload

Jetpack

WordPress公式の純正プラグイン。紹介する３大機能の他にも、様々な機能をひとまとめにしています。何を入れればいいかわからん！という人はとりあえず入れておくだけで快速化。

ダウンタイムのモニター

サイトがダウン（疎通不能）になると、メールアドレスに通知してくれる機能。自前でPingサーバーとか構築するのも手間がかかるので、プラグインで出来るのはGood。急なホスティング側の障害や、アクセス集中による鯖落ちを検知できます。

パフォーマンスおよびスピード （画像のクラウドキャッシュ高速化）

これが無料なのはすごいと思います。負荷になる画像ファイルをWordpress.comにキャッシュさせ、自分のサーバーでは軽いHTMLを配信するだけ。オススメ機能です。

アクセス解析

基本的なところですが。アクセス数、アクセス元、検索キーワードなどが表示できます。

Throws SPAM Away

SPAMフィルター。一番有名なのはAkismetだと思うんですが、これは実は無料版は「商用利用不可」なのです。まぁ使っちゃってる人も多いと思いますけど。

日本語を含まないコメントを禁止できます。SPAMの99割は海外からの投稿ですからね。

]]> Androidが動作重すぎたのにキャッシュ削除アプリで爆速化した件 http://seanworld.net:10214/original/gadget/32/ Fri, 13 Oct 2023 06:10:33 +0000 https://seanworld.net/?p=32 この記事はほぼすべてのAndroidで適用できる手順です。私の端末はXperiaですが(^^♪

なんかちょっと残念な見出しになってしまったけど、2年弱使っているXperiaが、Android 7.0アップデートぐらいから急激に重くなっていた。

LINEを立ち上げるのにも５秒、Chromeも５秒、と日に日にストレスがたまり続ける結果に。。。

最初は空き容量不足を疑いました。
しかし、画像や動画を消しても全然容量が増えない！「その他のアプリ」というジャンルに何十Gという容量を食われていました。

詳しく見てみると、例えば「LINE」のキャッシュだけで１G以上使われていたりしました。

ここで、「キャッシュを削除」を選択すると、あら不思議。１GB以上容量が空いて、スマホも速くなります。これを、Chrome、Gmail, Adobe Reader。。。などキャッシュを食うアプリにやっていけば、ある程度早くなります。

しかし、１か月後、そこには再びストレス発生マシーンと化したXperiaの姿が。。。

そこで、ストアを漁って見つけてきたのがこちらのアプリ。

ウイルス対策ソフトで実績のあるNorton社のアプリなので、ある程度は安心できます。アプリを開くとオレンジの画像みたいな画面が開くので、そこで黄色の風車ボタンを押すだけ。

これで、オンボロ（失礼）Androidが、嘘みたいに爆速になりますよ！！
