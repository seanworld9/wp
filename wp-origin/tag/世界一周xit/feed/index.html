世界一周20か国を回る上で、IT屋としては避けて通れないガジェット類の充電。今回は私の旅装から、世界一周回りきれる電源グッズを紹介していきます。

全世界対応マルチ電源プラグ　サスケ

本当に全世界に対応していました（神）。

シールに、どう組み合わせればどのタイプのプラグに刺さるようになるか書いてあります。

とりあえず私の行った国一覧に載っていたところは大丈夫。それ以外も多分大丈夫ですよ。かさばらないし、2つあってもいいぐらい。

ACアダプター( Anker PowerPort 5 )

Anker PowerPort 5 (40W 5ポート USB急速充電器) 【PSE認証済 / PowerIQ搭載】

他の記事で詳しくは書いていますが、充電対象ガジェットはかなり多い。（スマホメイン、スマホ予備、GoPro、Surface、モバブ×2)

今回買ったのは5ポートの製品ですが、宿では毎晩フル稼働してます。充電容量も申し分なし。今のところ、100V～240Vの国まで行きましたが、動作に影響なし。↓のSASUKE PLUGと組み合わせれば、あらゆる国のコンセントでスクスクとデバイスたちが充電されていっています。

https://www.amazon.co.jp/gp/product/B01GRHQ5US

ちなみに、旅を終えた後でも家でこのように活躍できるので、オススメ。現代の必需品ですね。猫も杓子もUSB充電。

モバイルバッテリー（Anker PowerCore 20100)

Surfaceなどパソコンレベルのガジェットの充電には、20000mAh級の容量がないと厳しいです。

街のスタバでちょっと旅レポでもSurfaceで書こうかなぁ～　って1日お休みムードで過ごす日だと、宿に帰るころには20000mAhでも空っぽになっています。

でも、せっかくの景色は、自分の目で確かめたほうが絶対いいです。魚眼で画角も広いので、前さえ向いていれば間違いなく収まっています。画面があると確認したくなっちゃうのが人情なので、いっそのこと無いほうがいいです。(一応スマホで確認可能）

3Way 自撮り棒

私のような一人旅行者なんかは、（しょうがなく）自撮りをすることも多いですよね。カップルとかでも、二人の思い出を残すにはこの自撮り棒がオススメ。

ちなみにあと1wayはグリップスタイルということで要するに3つ折りモードですね。短い棒なのでこれは割愛。

SIMフリースマホ1台で十分

初めに結論から言うと、SIMフリースマホを１台持っていけば、通信関係は問題ない。
iPhoneでもAndroidでも、できるだけ使い慣れているものがいいだろう。

現地のSIMカードは安い！

国によって1日300円～高くとも1000円程度。どの空港にもその国内で使えるSIMを売っている店が目立つところに必ずあるので、心配の必要はない。

SIMフリースマホの入手方法

「SIMフリー」とは？

どの携帯電話会社（世界中）のSIMカードを刺しても通信できるスマートフォンのことです。
日本で売っているスマホは、基本的に「SIMロック」されており、会社 (docomo, au, SoftBank)でしか使えないです( ；∀；)

中古のSIMフリースマホを購入

現在すでに中古スマホを買っている人、昔3大キャリアからスマホを買ったけれど今は格安SIMで使っている人などは、自分のスマホをSIMフリーにできないため、中古のケータイショップ（新品でももちろんOKだよ！高いけど！）で購入する必要がある。

世界一周の相棒
Xperia XZ　(22,000円)

安かったが、2019年現在も使い続けられている、非常にいい買い物でした。

２～３万円程度で、iPhone7 や Xperia XZ Premium が買えるので、このあたりの機種がおすすめです。

↓はオススメ中古ショップのイオシス。東京・大阪・名古屋など都心部には店舗が大体そろっている。
